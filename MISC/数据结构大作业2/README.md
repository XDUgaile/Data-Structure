- #### 注意事项
> 1. 为了降低耦合性，每种功能尽量写成函数
> 2. 声明变量尽量不要与其他函数的变量冲突
> 3. 环境尽量使用一致，都使用VS2017，编译器版本统一
> 4. 尽量都写成软代码，使用文件读取
